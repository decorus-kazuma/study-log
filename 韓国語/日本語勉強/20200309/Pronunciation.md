

## カ゜キ゜ク゜ケ゜コ゜、鼻濁音**

비탁음이라 불리는 이것은, ガ행이 특정한 환경에서 비음화하여 발음하는 것을 가리킨다. 어중, 어미의 ガ행은 비음화 시키며, 연탁화 된 것은 비탁음으로 된다. 그 외에도 조건이 있지만, 쉽게 보자면

| 조건                          | 여부 |
| ----------------------------- | ---- |
| 어중, 어미의 ガ행             | O    |
| 어두의 ガ행                   | X    |
| 외국어・외래어                | X    |
| 복합어에서 결합도가 강한 경우 | O    |
| 격조사・후속조사              | O    |
| 연탁된 단어                   | O    |
| 의성어・의태어                | X    |

가 있다.



일반적으로, 東日本〜東北이 ガ를 カ゜로 발음하나, 현대의 도쿄에서는 많이 사라졌다. 비탁음 예시의 대표적인 단어는 鏡가 있다.

| 단어         | 발음   | 비탁음   |
| ------------ | ------ | -------- |
| 鏡（かがみ） | カガミ | カカ゜ミ |
| 鍵（かぎ）   | カギ   | カキ゜   |



청자에게 더 부드럽게 들린다고 한다.
